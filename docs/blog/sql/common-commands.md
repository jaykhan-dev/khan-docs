# SQL

The most used language for databases, especially ones with relationships between tables. I'm most familiar with PostgreSQL but most SQL systems are virtually the same.

## `psql`

- `\du`: list users
- `\l`: list databases
- `\conninfo`: returns status of session
- `\c`: connect to a different database
- `\dt`: list tables

## Common SQL

The following commands and queries will work with any version of SQL like MySQL, PostgreSQL, SQLite etc.

```bash
CREATE DATABASE databaseName;
```

### Commands

- `SELECT` - extracts data from a database
- `UPDATE` - updates data in a database
- `DELETE` - deletes data from a database
- `INSERT INTO` - inserts new data into a database
- `CREATE DATABASE` - creates a new database
- `ALTER DATABASE` - modifies a database
- `CREATE TABLE` - creates a new table
- `ALTER TABLE` - modifies a table
- `DROP TABLE` - deletes a table
- `CREATE INDEX` - creates an index (search key)
- `DROP INDEX` - deletes an index
